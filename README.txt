Integration with the WriteAid content publishing system to remotely publish articles and content to your WordPress website directly from your WriteAid account.

== Description ==
WriteAid plugin provides a simple integration with the WriteAid content publishing system.

WriteAid is an efficient and streamlined article content and writer management solution. Content produced and submitted into the WriteAid system can be sent to your WordPress website(s) and published directly and easily from your WriteAid account.

For more information on the WriteAid platform and how you can manage your content writers with easy, visit us at [https://writeaid.net/](https://writeaid.net/).

== Installation ==
1. Create an account with WriteAid [here](https://writeaid.net/#pricing)
2. Once you have created an account, visit your [WordPress Integration](https://my.writeaid.net/settings/sites) area
3. Add your WordPress site and generate an API key that will be used in the plugin
4. Download and activate the WriteAid plugin from the WordPress plugin directory
5. On the WordPress plugin settings page, enter and validate your API key that was generated by WriteAid
6. Once validation is successful, you can now select your WordPress site when marking articles as Published in your Submissions area
